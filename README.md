### What I learned

I learned how to effectively use rems, ems, ch, and css custom properties.

```css
:root {
  --clr-white: hsl(0, 0%, 100%);
  --clr-light-grey: hsl(212, 45%, 89%);
  --clr-light-blue: hsl(220, 15%, 55%);
  --clr-dark-blue: hsl(218, 44%, 22%);

  --fw-body: 400;
  --fw-bold: 700;
}
```
